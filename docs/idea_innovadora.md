## 🛒 Lista de Compras Inteligente
Descripción

Lista de Compras Inteligente es una aplicación desarrollada en Python que permite gestionar productos de manera sencilla y eficiente. A diferencia de una lista de compras tradicional, el sistema almacena el historial de compras y analiza los productos más frecuentes para generar recomendaciones automáticas, ayudando al usuario a planificar futuras compras y ahorrar tiempo.

Características
Agregar, editar y eliminar productos.
Marcar productos como comprados.
Guardar historial de compras.
Generar sugerencias basadas en compras anteriores.
Calcular el costo total estimado de la lista.
Exportar información a archivos CSV o JSON.
Control de versiones mediante Git y GitHub.
Innovación del Proyecto

La principal innovación consiste en incorporar un sistema de recomendaciones que aprende de los hábitos de compra del usuario. A medida que se registran compras, la aplicación identifica patrones y sugiere automáticamente productos que podrían ser necesarios nuevamente.
