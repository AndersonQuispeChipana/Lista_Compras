# 🛒 Lista de Compras

## 📖 Descripción
Aplicación desarrollada en Python que permite gestionar una lista de compras de forma sencilla. Los usuarios pueden agregar, eliminar, modificar y visualizar productos, facilitando la organización de sus compras diarias.

## ✨ Características
- Agregar productos a la lista.
- Eliminar productos.
- Editar productos existentes.
- Visualizar todos los productos registrados.
- Guardar y cargar datos automáticamente.
- Interfaz por consola fácil de usar.
