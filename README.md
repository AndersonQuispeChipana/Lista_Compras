# 🛒 Lista_Compras 

## 📌 Descripción

`Lista_Compras Inteligente` es una aplicación desarrollada en Python que permite gestionar y organizar listas de compras de manera rápida y eficiente desde la terminal.

El sistema ayuda a los usuarios a registrar productos, clasificarlos por categorías y controlar los gastos de sus compras.

---

# 🎯 Objetivos del proyecto

## Objetivo principal

Desarrollar un sistema informático en Python para administrar listas de compras de forma organizada y sencilla.

## Objetivos específicos

* Agregar productos
* Eliminar productos
* Editar productos
* Clasificar productos por categorías
* Guardar información en archivos
