# Lista de Compras 🛒

Una aplicación en Python para gestionar tu lista de compras personales

## 📝 Descripción

Proyecto que permite agregar, eliminar y visualizar productos en una lista de compras, 
con opciones para categorizar y calcular el total estimado.

## 🚀 Características

- ✅ Agregar productos con nombre y precio
- ✅ Eliminar productos de la lista
- ✅ Visualizar toda la lista actualizada
- ✅ Calcular el total estimado de la compra
- ✅ Categorizar productos (alimentos, bebidas, limpieza, etc.)
- ✅ Guardar y cargar lista desde archivo (JSON/TXT)

## 🛠️ Tecnologías utilizadas

- **Python**
- **GitHub**: Para control de versiones
